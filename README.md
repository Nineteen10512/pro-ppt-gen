# Pro PPT Gen v1.6.5

专业级 PPT 生成技能，基于 PaperJSX 语义编译架构，从 Markdown/JSON 直接生成可编辑 `.pptx` 文件。

- 🎨 **可视化重设计内核**：防平铺质量门，自动规避"Word 文档上色"式幻灯片
- 📐 **信息图结构要素**：核心页强制连接线/箭头/节点/层级等 ≥2 类结构要素
- 🏗️ **50+ 行业模板**：学术答辩/教学课件/商业路演/产品发布/年终总结
- ⭐ **评分系统** + 模板智能推荐
- 📦 原生 `.pptx` 输出，无需中间格式转换
