# Pro PPT Gen v1.6.5

> AI-powered presentation generator — visual redesign engine, anti-flat quality gates, 50+ industry templates.

**Pro PPT Gen** 是一款专业级 AI PPT 生成技能，支持多种视觉风格、模板复刻、自动重试和 HTML/PPTX 导出。v1.6.4 起引入可视化重设计内核与防平铺质量门，彻底告别「文字堆砌」式幻灯片。
